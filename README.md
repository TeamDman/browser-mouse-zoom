# Right Mouse Zoom

This browser extension lets you zoom the current page by scrolling while holding the right mouse button.

## What it does

- Hold the right mouse button.
- Scroll up to zoom in.
- Scroll down to zoom out.

The extension changes the tab zoom directly, so it behaves like browser page zoom rather than CSS scaling.

## Install in Chromium browsers

1. Open `chrome://extensions` or `edge://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this folder.

## Notes

- The first scroll gesture while right-click is held suppresses the context menu so the gesture feels natural.
- Some browser-internal pages, extension pages, or restricted URLs will not allow content scripts or zoom changes.
